# Akira

Projeto ETL para processamento de dados.

## Estrutura do Projeto

- `data/`: Diretório com dados brutos (bronze), tratados (silver) e processados (gold)
- `etl/`: Scripts de extração, transformação e carregamento
- `main.py`: Ponto de entrada principal

## Requisitos

- Python >= 3.12
- UV (gerenciador de pacotes)

## Instalação

```bash
uv sync
```

## Uso

```bash
python main.py
```
